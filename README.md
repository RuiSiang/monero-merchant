# monero-merchant
![](https://github.com/RuiSiang/monero-merchant/workflows/Node.js%20CI/badge.svg)
Nodejs Merchant for Monero

## Features

* Multiple payment modes: receive only, finalize early, escrow
* Different address formats: subaddress, integrated address
* Friendly API
